# Calculation of Pi and Circumference

## Goals

To calculate the circumference of sun with precise Pi value.

## Limitations

The is no database and storage used. 

## Run the project locally

### Run the server

```
cd server
npm install
node app.js
```

### Run the client

```
cd client
npm install
npm run dev
```
